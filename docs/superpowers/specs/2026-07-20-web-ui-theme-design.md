# Web UI 明暗主题设计

## 目标

为本机 Codex Gateway Web UI 增加太阳/月亮主题按钮，在不改变现有布局与功能的前提下提供完整暗色主题。

## 交互

- 首次访问时跟随系统 `prefers-color-scheme`。
- 浅色主题显示月亮按钮，点击切换到暗色主题。
- 暗色主题显示太阳按钮，点击切换到浅色主题。
- 用户选择写入 `localStorage`，后续访问优先使用该选择。
- 按钮提供随目标主题变化的 `aria-label` 与 `title`。

## 实现

- 在页面 `<head>` 的样式加载前执行极小的主题初始化脚本，提前设置根节点 `data-theme` 和 `color-scheme`，避免暗色用户看到白色闪屏。
- 将现有硬编码界面颜色收口为 CSS 变量，并用 `:root[data-theme="dark"]` 覆盖变量。
- 页面主脚本负责按钮图标、提示文本、主题切换和持久化。
- 不引入图标或前端框架依赖，使用系统可识别的太阳/月亮字符。

## 验证

- 页面包含暗色变量、主题按钮、系统主题检测与本地持久化逻辑。
- 所有内嵌脚本均可编译。
- 原有 Web API 和监控页面测试保持通过。
